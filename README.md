# cse-project-vityarthi
Personal Finance Manager

The 
Personal Finance Manageris a simple Python project designed to help users track their day-to-day income and expenses. It provides a basic system where users can record money coming in, money going out, and automatically calculate the remaining balance. This project is especially useful for students, beginners in Python, and anyone who wants a small, easy-to-understand financial tool.

# Features
- Add income with description  
- Add expense with description  
- Automatically calculates total income, expenses, and balance  
- Simple text-based interface  
- Beginner-friendly code structure
